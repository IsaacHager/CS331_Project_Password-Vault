Password Manager

Quick CLI Equivalent: java UserLoginUI <add|login> <username> <password> add - create a new user login - check a user’s credentials username: letters, numbers, _ (4–16 chars) password: letters, numbers, symbols (8–16 chars)
Choose what you want to do (Login or Add User).
4–16 characters: letters, numbers, underscore.
8–16 characters; symbols like !@#$%^&*() work too.
