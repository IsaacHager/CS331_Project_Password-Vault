Password Manager

Command Line Usage Equivalent: java UserLoginUI <add|login> <username> <password> Where: Add - Adds a user to the database Login - Verifies a user against the database Username - [A-Za-z0-9_]{4,16} Password - [A-Za-z0-9_!@#$%^&*()]{8,16}
Pick whether you're adding or logging in.
Length of 4–16 characters: letters, numbers, underscore.
Length of 8–16 characters. Special chars allowed: !@#$%^&*()
