Password Manager

Usage (CLI equivalent): java UserLoginUI <add|login> <username> <password> where: add - adds a user to the database login - verifies a user against the database username - [A-Za-z0-9_]{4,16} password - [A-Za-z0-9_!@#$%^&*()]{8,16}
Must be either be add or login.
Allowed: letters, digits, and underscore. Length of: 4–16.
Allowed: letters, digits, underscore, and !@#$%^&*(). Length of: 8–16.
